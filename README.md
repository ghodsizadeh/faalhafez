# فال حافظ | Hafez Faal

## فارسی

این پروژه یک صفحهٔ وب سبک و ساده برای گرفتن «فال حافظ» است. با یک کلیک می‌توانید یک غزل تصادفی ببینید یا با وارد کردن شمارهٔ غزل، همان غزل را نمایش دهید. همچنین امکان انتخاب چند غزل و چاپ هم‌زمان آن‌ها هم وجود دارد.

هدف اصلی این پروژه یک استفاده شخصی برای پرینت کردن چند فایل بود. این روزها با هوش مصنوعی ساختن صفحات وب راحتتر از کار کردن با ورد است!
### امکانات

- **فال تصادفی**
- **انتخاب غزل با شماره**
- **اضافه کردن چند غزل به لیست** و **چاپ یکجای همه**

### اجرا

- نسخهٔ آنلاین:
  - https://ghodsizadeh.github.io/faalhafez/
- اجرای محلی (اختیاری): فایل `index.html` را در مرورگر باز کنید.

نکته: برای بارگذاری سریع‌تر، داده‌ها در `faal-data.js` به صورت `gzip + base64` فشرده شده‌اند و در مرورگر باز می‌شوند (پیچیده‌ترین بخش کار!).

### منبع داده‌ها و قدردانی

داده‌های فال (غزل‌ها) از مخزن زیر تهیه شده است:

- [HafezFaalDatabase](https://github.com/mahmoud-eskandari/HafezFaalDatabase)

با تشکر از **محمود اسکندری** برای جمع‌آوری و به اشتراک‌گذاری این مجموعه ارزشمند.

## English

This project is a lightweight single-page web app for Hafez fortune-telling (Faal-e Hafez). With one click you can get a random ghazal, or you can type a ghazal number to view a specific one. It also supports selecting multiple faals and printing them together.

The main goal of this project is to print multiple files. These days it is easier to create web pages with AI than to work with Word! 
### Features

- **Random faal**
- **Pick a ghazal by number**
- **Build a list** of faals and **print them all at once**

### Run

- Online version:
  - https://ghodsizadeh.github.io/faalhafez/
- Local (optional): open `index.html` in your browser.

Note: For faster loading, the data is stored in `faal-data.js` as `gzip + base64` and decompressed in the browser (the most complex part!).
